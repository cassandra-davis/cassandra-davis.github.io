Predictive Maintenance Dashboard

Data-driven insights for fault prediction in automotive systems — helping you act before issues occur.

Tools Used

Python, pandas, matplotlib, Power BI

About the Project

This dashboard was developed to identify fault patterns within test and sensor data, enabling earlier interventions and more reliable system behaviour. The aim was to turn raw datasets into meaningful insights that support engineering decisions.

Process

The work included cleaning and transforming large datasets, building predictive indicators, and visualising the results through both Python and Power BI. The dashboard was structured to highlight trends, anomalies, and early-warning signs in a clear and actionable way.

Deliverables

  • Interactive Power BI dashboard
  • Python scripts for data ingestion and feature extraction
  • Visual reports showing fault likelihood and trends

Results

The solution improved detection of potential issues ahead of time, reducing unplanned downtime and improving reliability across test cycles. Engineers gained clearer visibility into failure patterns, leading to more targeted and efficient investigations.
