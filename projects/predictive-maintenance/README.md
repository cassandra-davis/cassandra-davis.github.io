# Predictive Maintenance Dashboard

## Overview
Simulated automotive fault prediction dashboard. Analyses sensor data to predict component failures and visualises trends for proactive maintenance.

## Tools Used
Python, pandas, matplotlib, Power BI (optional for dashboard)

## Steps
1. Generate synthetic telemetry data (`predictive_data.csv`)
2. Analyse trends and correlations in Python notebook (`predictive_maintenance.ipynb`)
3. Create visualisations for failures
4. Optional: Export plots/dashboard to HTML or Power BI

## Business Value
Early fault detection reduces downtime, improves reliability, and helps teams prioritise maintenance proactively.
